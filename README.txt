Inventory System (Blazor Server, .NET 8)

1. Open InventorySystem.sln in Visual Studio (ensure ASP.NET workload + .NET 8 SDK installed).
2. Restore NuGet packages.
3. Run DatabaseScript.sql in SSMS to create the InventoryDB (or change connection string in appsettings.json).
4. Set Inventory.Web as the startup project, build and run.
Default admin credentials: admin / 123 (plain text password as requested).
